# Code Of Conduct

This project follows the Contributor Covenant 2.1 standard.

## Expected Behavior

- Be respectful and direct.
- Keep technical disagreement focused on code, docs, and project goals.
- Assume public issues and PRs are permanent project records.
- Help keep the project welcoming for new contributors.

## Unacceptable Behavior

- Harassment, insults, threats, or sustained disruption.
- Sexualized language or imagery.
- Publishing private information without permission.
- Dismissing people based on identity, background, or experience level.

## Enforcement

Maintainers may edit, hide, or delete comments; close issues or PRs; block
participants; or report abuse to GitHub.

Report conduct concerns through GitHub. If public reporting would expose private
details, ask a maintainer for a private contact channel first.
